# Mini Flask Project

## 📌 Project Overview

This project is a simple Flask IAP with Swagger and Sphinx documentation.
It demonstrates how to build REST APIs, document them using Swagger, and generate project documentation using Sphinx.

---

## 🚀 Features

- Flask REST API
- Swagger API Documentation
- Sphinx Documentation
- GitHub Pages Documentation
- Python Virtual Environment

---

## 🛠 Tech Stack

- Python
- Flask
- Flasgger (Swagger)
- Sphinx
- GitHub Pages

## ⚙️ Installation

### 1. Clone repository

git clone https://github.com/Tung867/miniproject8.git

cd mini-flask-project

---

### 2. Create virtual environment

python -m venv venv

venv\Scripts\activate

---

### 3. Install dependencies

pip install -r requirements.txt

---

### 4. Run Flask

python run.py

Server will run at:

http://127.0.0.1:5000
